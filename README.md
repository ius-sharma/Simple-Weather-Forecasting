# Dynamic Weather Dashboard (TypeScript)

A Simple weather dashboard built using HTML, CSS, and TypeScript.

This project demonstrates real-time API integration, async/await handling, dynamic geocoding, and clean UI architecture.

---

## 🚀 Features

- 🌍 Dynamic city search (All cities supported)
- 🔗 Geocoding API integration
- 🌦 Live weather data via Open-Meteo API
- 🎨 Weather-based UI updates
- ⚡ Async/Await with proper error handling
- 💎 Clean responsive dashboard layout.

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox layout)
- TypeScript
- Fetch API
- Open-Meteo Weather API

---

## 🧠 Key Concepts Implemented and The thing which learned

- API chaining (Geocoding → Weather API)
- Modular TypeScript structure
- DOM manipulation with type safety
- Error handling & loading state management
